Customize the WooCommerce My Account page: reorder the menu, add custom endpoints, groups and links, set per-endpoint content and control visibility by user role.

== Description ==

My Account Customizer for WooCommerce lets you take full control of the "My Account" area:

* Reorder the account menu and toggle any item on or off.
* Add custom endpoints with their own URL and content.
* Add groups (collapsible parents) and external/internal links.
* Insert custom content per endpoint (before, after or replacing the default).
* Show or hide items based on user role.
* Restyle the menu: position (vertical/horizontal), layout and colors.
* Optional AJAX navigation between endpoints.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate through the 'Plugins' screen in WordPress.
3. Go to WooCommerce → My Account to configure.
